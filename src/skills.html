Instalación

Los skills se instalan copiando la carpeta skills/ del repositorio de holygrailcss en el directorio .claude/skills/ de tu proyecto. Claude los detectará automáticamente.

1

Clonar o copiar los skills

Desde el repositorio de holygrailcss, copia la carpeta skills/

# Copiar skills al proyecto cp -r node_modules/holygrailcss/skills/ .claude/skills/holygrailcss/
2

Verificar la estructura

Tu proyecto debería tener esta estructura:

.claude/skills/holygrailcss/ ├── component-generator/SKILL.md ├── config-builder/SKILL.md ├── theme-creator/SKILL.md ├── migration-helper/SKILL.md └── layout-builder/SKILL.md
3

Usar en Claude

Claude detectará los skills automáticamente. Pide cosas como:

# Ejemplos de uso "Genera un componente card de producto con holygrailcss" "Crea un config.json para mi proyecto e-commerce" "Crea un tema oscuro basado en Black&Yellow" "Migra mis clases de Bootstrap a holygrailcss" "Construye un layout de landing page responsive"

Component Generator

skills/component-generator

Genera componentes HTML que aprovechan el sistema completo de utilidades de holygrailcss. Lee el config.json del proyecto para usar solo clases que existen.

  • Valida clases contra el config.json real del proyecto
  • Usa spacing, helpers, grid, tipografía y aspect ratios
  • Genera marcado mobile-first con variantes responsive (md\:)
  • Incluye comentarios explicando las decisiones de layout
  • Verifica que colores y spacing referenciados existen

Config Builder

skills/config-builder

Asiste en la creación o modificación del config.json, validando la estructura completa y asegurando coherencia entre tokens.

  • Valida propiedades obligatorias (prefix, breakpoints, typo)
  • Genera colores, spacing, tipografía, helpers y grid
  • Verifica consistencia entre breakpoints y valores
  • Ejecuta build para validar que genera CSS sin errores
  • Detecta duplicados en spacingMap, colors y typo

Theme Creator

skills/theme-creator

Crea temas personalizados siguiendo la estructura ITCSS del tema Black&Yellow. Genera variables, componentes CSS modulares y demo interactiva.

  • Estructura ITCSS: Settings → Components → Utilities
  • Variables del tema extienden las de holygrailcss (--hg-color-* → --btn-*)
  • Genera _buttons, _inputs, _checkboxes, _radios, _switches, _forms
  • Crea demo.html con secciones navegables por componente
  • Activa automáticamente el tema en config.json

Migration Helper

skills/migration-helper

Asiste en la migración desde otros frameworks CSS. Analiza el código existente, mapea clases y genera un plan de migración.

  • Mapas de equivalencia: Bootstrap → holygrailcss, Tailwind → holygrailcss
  • Analiza CSS actual para extraer tokens (colores, spacing, tipografía)
  • Genera config.json a partir del análisis del proyecto
  • Plan de 7 pasos: instalar → config → generar → enlazar → reemplazar → limpiar
  • Buscar y reemplazar clases en lote

Layout Builder

skills/layout-builder

Construye layouts de página completos combinando grid, flexbox, spacing responsive y aspect ratios.

  • Grid de 12-24 columnas con 5 breakpoints (xs → xl)
  • Patrones: hero, grid de cards, sidebar layout, split content
  • Helpers de flexbox para alineación dentro de componentes
  • Aspect ratios para imágenes y media
  • Verifica que no hay overflow horizontal en mobile
