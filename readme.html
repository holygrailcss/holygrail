holygrail

Holygrail is a modern CSS framework based on Flexbox.

Quick install

holygrail is constantly in development! Try it out now:

NPM

npm install holygrail (not yet)

or

Yarn

yarn add holygrail

Bower

bower install holygrail

Import

After installation, you can import the CSS file into your project using this snippet:

import 'holygrail/css/holygrail.css'

CDN

https://cdn.jsdelivr.net/gh/holygrailcss/holygrail/dist/style.css

Feel free to raise an issue or submit a pull request.

CSS only

holygrail is a CSS framework. As such, the sole output is a single CSS file: holygrail.css

You can either use that file, “out of the box”, or download the Sass source files to customize the variables.

There is no JavaScript included. People generally want to use their own JS implementation (and usually already have one). holygrail can be considered “environment agnostic”: it’s just the style layer on top of the logic.

Browser Support

holygrail uses autoprefixer to make (most) Flexbox features compatible with earlier browser versions. According to Can I use, holygrail is compatible with recent versions of:

  • Chrome
  • Edge
  • Firefox
  • Opera
  • Safari

Internet Explorer (10+) is only partially supported.
